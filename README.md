**MARGIN**
**oque é:**
A propriedade **_margin_** no CSS é utilizada para criar espaçamentos externos ao redor de um elemento, separando-o de outros elementos na página. Ela é uma ferramenta essencial para ajustar layouts e posicionar elementos de forma precisa.

**-exemplo:**
margin: 10px;

**BORDER**
**oque é:**
No **CSS**, a propriedade **`border`** é usada para definir a **borda** de um elemento HTML.   Ela controla **espessura**, **estilo** e **cor** da borda.

**-exemplo:**
border: 1px  solid  black;

**PEDDING**
**oque é:**
No CSS, **padding** é o **espaçamento interno** de um elemento — ou seja, a distância entre o conteúdo (texto, imagem, etc.) e a borda do elemento.

**-exemplo:**
padding: 10px;
